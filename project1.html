PROJECT 1:
Portfolio website design

Project Reflection

The primary purpose of this website is to function as a scalable portfolio website for my coding projects. I wanted my website to be clean, stylish and easy to understand and navigate. I believe I have achieved this goal.

Along the way I have learned so much about web design and workflow, I now know how to use Github for version control. I can implement flexbox to create responsive design on my websites so my pages can seamlessly adapt to different screen sizes. I know how to add images, style text, change colors and much more.

When I was designing this website I wanted it to feel modern and futuristic but also immersive and atmospheric, to achieve this I added a background image of a forest to give a natural theme to the website, I wanted the background to still be visible under my content but also not overwhelming the user so I used a style called ‘Glassmorphism’ which involves making elements on the page semi-transparent and blurring the background behind them to give the illusion of looking through blurry glass, this style is often used by companies like apple.

I used futuristic fonts and a grey color palette to make the pages feel more ‘sci-fi’ without going overboard on glow and shadows to keep it simple. I was very intentional about what colors I used for specific elements to draw the users focus where I wanted it like the intro box which I made contrast with the other elements of the page, when the user hovers their cursor over links on the page they increase in size and change color which gives the website more interactivity.

I made sure to future-proof my homepage so it will support the additional pages and content I plan to add in future, I did this by creating several ‘dummy’ boxes and pages that can be assigned to new content whenever I need, i also created a template HTML page so when i do need to add more pages i can just paste my content in without all the headache of having to rebuild anything.

Overall, I am proud of the design and functionality of my portfolio website. It effectively showcases my coding projects and skills, while also being visually appealing and easy to navigate. I believe it will be a valuable tool for me as I continue to develop my coding skills and build my portfolio.
